# jsmpeg-player-demo

## JSMpeg 对象引入

文件 [public/index.html](public/index.html) 第15行，引入JSMpeg对象：

```html
<script src="<%= BASE_URL %>jsmpeg.min.js" charset="utf-8"></script>
```

文件 [jsmpeg.min.js](public/jsmpeg.min.js) : `public/jsmpeg.min.js`

## 播放器 svg icon

文件位于  `public/player/` 目录下
